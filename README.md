# CareMatch Prototype

Mobile-first React + TypeScript + Vite mockup for a senior-care matching product.

## Stack

- React 19
- Vite 7
- TypeScript (strict)
- Tailwind CSS v4 via `@tailwindcss/vite`
- Base UI (`@base-ui/react`) for accessible primitives
- shadcn-style local UI wrappers (`Button`, `GlassCard`, `Badge`, `GlassDialog`)

## Run

```bash
npm install
npm run dev
```

## Notes

- The map is intentionally a visual prototype rather than a real GIS integration.
- The discovery flow mixes **map context** with **swipe-inspired cards**.
- Styling direction: **Apple-ish liquid glass**, soft highlights, frosted panels, calm care-product tone.
